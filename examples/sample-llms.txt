# FastAPI

> A platform providing FastAPI, modern, web and related resources.

Key information available on this site:

- FastAPI - modern, fast (high-performance), web framework for building APIs with Python 3.7+ based on standard Python type hints.
- Learn - Step by step tutorials, featuring all the features, from beginner to advanced.
- User Guide - The user guide has all you need to know to use FastAPI.
- Reference - Technical reference, API, parameters, body, etc.
- Deployment - Several easy ways to deploy your FastAPI application.
